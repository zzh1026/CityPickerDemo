#本 Demo 的主要功能

### 1,正式第一次使用 git　在本地开发连接　github　上传 ###
### 2,CityPicker原来的项目中的字体显示不居中，偏下，而且 onDraw　的时候字体是从左到右的，现在修改了　通过之前对　自定义　drawText　的学习，使得字体完全居中，主要处理了两个点 ###
	- 字体的描绘方式为从中间到两边绘制，这样保证了竖直居中．
	- 字体位置通过字体的头部和底部进行了判定修改，保证了水平居中．
